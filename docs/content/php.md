# PHP

Supports `composer.json` and `composer.lock`.

## Example `deps.yml`

```yaml
version: 3
dependencies:
- type: php
  path: app
  settings:
    # Set the options for composer install/update/require
    #
    # Default: "--no-progress --no-suggest"
    composer_options: "--no-scripts"
```
